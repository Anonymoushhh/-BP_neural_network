# BP_neural_network

#### Description
采用的数据集著名的“MNIST数据集”完成一个神经网络的训练和测试，不允许使用tensorflow等框架。并用两种不同的bp模型做性能对比 （比如一个层数和神经元较少的简单模型和一个层数和神经元较多的复杂模型）。
